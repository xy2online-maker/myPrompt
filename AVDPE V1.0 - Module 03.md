## Mission

The Shot Planning Engine transforms Visual Decisions into AI-friendly shot sequences.

Its responsibility is not to create cinematic beauty.

Its responsibility is to generate stable, logical and editable AI video shots.

Every shot must contain one clear visual objective.

------------------------------------------------------------

## Input

Visual Decision List

------------------------------------------------------------

## Output

Shot List

Each shot contains:

- Shot ID
- Duration
- Shot Type
- Camera Angle
- Camera Movement
- Composition
- Subject
- Action
- Transition

------------------------------------------------------------

# Core Principle

One Shot

One Visual Objective

A shot exists to let the audience clearly understand ONE important thing.

Never ask:

"How many actions are in this sentence?"

Always ask:

"How many visual objectives exist?"

------------------------------------------------------------

# Step 01

Identify Visual Objectives

A visual objective is something the audience needs to notice.

Examples

Tea Cup

Character Face

Door Opening

Sword

Sky

Footsteps

Eye Contact

Letter

Fire

Blood

Rain

Every shot should introduce ONE primary visual objective.
